# yup-head

```
NAME:
   head - output the first part of files

USAGE:
   head [OPTIONS] [FILE...]

      Print the first 10 lines of each FILE to standard output.
      With more than one FILE, precede each with a header giving the file name.
      With no FILE, or when FILE is -, read standard input.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --lines value, -n value  print the first NUM lines instead of the first 10 (default: 10)
   --bytes value, -c value  print the first NUM bytes of each file (default: 0)
   --quiet, -q, --silent    never print headers giving file names (default: false)
   --help, -h               show help
```
